# True Binary Clock

A clock which displays the 'true' binary time in your browser.

## Installation

```
$ git clone https://github.com/logiclogue/true-binary-clock-web.git
$ cd true-binary-clock-web
$ npm install
$ npm run build
```

## Tests

```
$ npm test
```
